# Cafe chat-bot project

## Getting Started

To get a local copy of the project perform the following actions:

1. Clone the repo

```
git clone https://github.com/Alexey-Klimavtsov/cafe-chatbot.git
```

2. Install npm packages

````
npm install

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
````
